How To Use OpenARMS
01 DiscoverScan the laptop LAN, pick the ARM nodes, assign controller/compute/storage roles, then use the selected nodes.
02 PrepareRun prereq scan, bootstrap nodes, and generate the Kolla-Ansible inventory from the selected targets.
03 DeployChoose Kolla-Ansible or Genestack, set the OpenStack release, VIP, and external interface, then deploy when the lab is ready.
04 ValidateRun validation to test ARM64 image boot, Neutron networking, floating IPs, Cinder/LVM, and serial console access.
05 ImproveUse Result Stage to read logs, Hardware & Software Fix to suggest changes, then retest from Stage 01.
